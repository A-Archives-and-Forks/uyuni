 34      """  
 35      Convert a query with named parameters (i.e. :id, :name, etc) into one 
 36      that uses %(id)s, %(name)s parameters instead. 
 37   
 38      python-psycopg2 requires parameters to be in this form, so to keep our 
 39      existing queries intact we'll convert them when provided to the  
 40      postgresql driver. 
 41   
 42      RETURNS: tuple with: 
 43          - the new query with parameters replaced 
 44          - hash of each named parameter to an ordered list of the positions 
 45            where it was used. 
 46          - number of arguments found and replaced 
 47      """ 
 48      log_debug(6, "Converting query for PostgreSQL: %s" % query) 
 49      new_query = re.sub(r'(\W):(\w+)', r'\1%(\2)s', query.replace('%', '%%')) 
 50      log_debug(6, "New query: %s" % new_query) 
 51      return new_query
