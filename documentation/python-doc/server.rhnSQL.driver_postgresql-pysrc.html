 39      """  
 40      Convert a query with named parameters (i.e. :id, :name, etc) into one 
 41      that uses %(id)s, %(name)s parameters instead. 
 42   
 43      python-psycopg2 requires parameters to be in this form, so to keep our 
 44      existing queries intact we'll convert them when provided to the  
 45      postgresql driver. 
 46   
 47      RETURNS: tuple with: 
 48          - the new query with parameters replaced 
 49          - hash of each named parameter to an ordered list of the positions 
 50            where it was used. 
 51          - number of arguments found and replaced 
 52      """ 
 53      log_debug(6, "Converting query for PostgreSQL: %s" % query)

 56      return new_query
